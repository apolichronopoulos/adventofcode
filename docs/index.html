[Custom Leaderboard 2024]

This year, to make the challenge a bit more compelling, we introduced a custom set of rules to calculate the leaderboard scores.

  • For each star (2 per day) you get points equal to the number of the day (2 points for day 1)
  • Extra 1 point if you finish the puzzle in the same day it's released (50% for each star)
  • Extra 1-5 points if you’re among the first five people to solve the puzzle (50% for each star)
  • Some extra points are going to be added if you are using a less popular/more difficult programming language (check here)
